# 项目概述

## 1. 项目名称

基于多模态大模型的设备检修知识检索与作业辅助系统

## 2. 项目背景

设备检修场景长期存在知识分散、经验依赖强、问题响应慢、标准化不足等问题。维修手册、故障案例、作业规范和安全要求通常分散在 PDF、文档、图片和人工经验中，一线人员难以在故障现场快速获取可靠答案。

本项目围绕比赛要求，建设一个面向设备检修场景的智能辅助系统，利用大模型和知识检索能力，将资料输入、语义检索、检修问答、作业指导、工单记录串联为一条完整链路。

## 3. 项目目标

- 建设统一的检修知识入口，支持设备资料集中管理。
- 让用户通过自然语言快速获取故障排查建议。
- 输出结构化的检修步骤、安全提醒和依据来源。
- 将问答结果沉淀为检修记录，支撑复盘和答辩演示。

## 4. 目标用户

- 制造业或设备运维场景的一线检修人员。
- 新入职或经验不足的现场维护人员。
- 需要查看检修记录和作业过程的管理人员。

## 5. 核心能力

- 文档上传与知识入库。
- 基于知识库的检修问答。
- 标准化作业步骤输出。
- 安全注意事项提示。
- 检修记录生成与查看。

## 6. MVP 主链路

设备资料上传 -> 文档解析与切分 -> 向量检索 -> 检修问答 -> 作业指导 -> 工单记录

## 7. 当前建设策略

项目第一阶段不追求复杂权限系统、知识图谱、复杂多 Agent、视频分析等重功能，而是优先保证最小闭环可运行、结构清晰、文档完备、便于后续扩展。
