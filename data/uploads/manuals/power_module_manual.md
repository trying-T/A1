# 电源模块维修手册

## 1. 电源模块启动异常

当设备启动后电源指示灯不亮，风扇也不转时，应优先排查供电输入链路。常见原因包括外部电源未接入、交流输入异常、直流适配器损坏、保险丝熔断、急停开关未复位、供电线束松动、电源开关损坏或电源模块进入保护状态。

排查步骤如下：
1. 确认外部电源插座或供电端子是否有电。
2. 检查电源线、插头、接线端子是否松动或破损。
3. 检查设备急停开关是否处于按下状态。
4. 检查电源保险丝是否熔断。
5. 检查电源开关输入端和输出端电压是否正常。
6. 若输入正常但电源指示灯仍不亮，应检查电源模块内部保护或损坏情况。

在未确认断电前，不得拆开电源模块外壳。

## 2. 风扇不转故障

风扇不转可能由供电异常、风扇接口松动、风扇电机损坏、温控信号异常或主控板未输出风扇控制信号引起。若电源指示灯不亮且风扇不转，应优先排查整机供电；若电源指示灯正常但风扇不转，应检查风扇供电端、电源接口和控制信号。

## 3. 运行一段时间后自动断电

设备运行 10 分钟后自动断电，常见原因包括电源模块过热保护、负载过大、散热风扇失效、散热孔堵塞、内部短路、电源输出不稳定或温度传感器异常。

排查步骤如下：
1. 检查设备运行环境温度是否过高。
2. 检查风扇是否正常转动。
3. 检查散热孔和滤网是否被灰尘堵塞。
4. 检查电源模块表面是否明显发热。
5. 检查负载设备是否存在短路或过载。
6. 等待设备冷却后重新上电，观察是否再次断电。

若设备反复自动断电，应停止继续运行，避免电源模块损坏扩大。

## 4. 过温报警

当设备出现过温报警时，应检查散热风扇、散热片、通风孔、滤网、环境温度、温度传感器和电源模块负载情况。过温报警通常说明设备内部温度超过安全阈值，继续运行可能导致元件老化、绝缘性能下降或电源保护动作。

处理建议：
1. 立即降低负载或停止设备运行。
2. 检查风扇是否运转。
3. 清理散热孔和滤网。
4. 检查温度传感器连接线是否松动。
5. 确认设备周围是否有遮挡物影响散热。

## 5. 报警代码 E07

报警代码 E07 表示电源模块过温保护或散热异常。出现 E07 时，应优先检查风扇运行状态、散热通道、滤网堵塞情况、电源模块温度和环境温度。如果风扇不转或散热孔堵塞，应先处理散热问题，再重新上电测试。
